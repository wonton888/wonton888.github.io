Pig Dice

A pig dice game app using javascript and jquery. The goal of the pig dice game is to score 100 points before your opponent does using a six-sided die. You can continue rolling the die until you are satisfied with your score, but if you roll a one you lose all the points accumulated and lose a turn. Check out the Live Demo or check out the code in the project's Github Repository.
